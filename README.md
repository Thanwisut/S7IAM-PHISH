## 🛠️ วิธีติดตั้ง

> แนะนำให้รันใน Termux หรือ Linux (Debian/Ubuntu-based)

### 📦 ติดตั้งบน Termux

```bash
pkg update && pkg upgrade -y
pkg install git php openssh -y
git clone https://github.com/Thanwisut/S7IAM-PHISH
cd S7IAM-PHISH
chmod +x start.sh
./start.sh
```
